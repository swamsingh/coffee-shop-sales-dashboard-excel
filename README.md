Coffee Shop Sales Dashboard (Excel)

This project demonstrates an interactive Excel dashboard built using a public coffee shop sales dataset.

Key highlights:
- Sales KPIs and customer metrics
- Hour-wise and weekday sales analysis
- Product category and size distribution
- Store-wise performance comparison
- Interactive slicers for month and day filtering

Tools Used:
- Microsoft Excel
- Pivot Tables & Pivot Charts
- Slicers
- Data Cleaning

Note:
This project uses sample data only and does not include any confidential or proprietary company information.
